# PDF Service

This project generates a PDF from JSON data using Node.js, Puppeteer, React, and Tailwind CSS.

It includes:
- a backend API to generate/download PDFs
- a simple frontend form to enter data
- a fixed PDF template (same style for every generated file)

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:4000` and fill the form to generate a PDF.
